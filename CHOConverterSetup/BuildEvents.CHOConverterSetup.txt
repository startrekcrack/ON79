Build Process:
  1. BuildCHOConverter target builds all three projects (CLI, GUI, WPF)
  2. CombineCHOConverterOutputs target combines all outputs into one directory
  3. GenerateHarvest target uses Heat.exe to harvest all files
  4. Remove-PdbFromHarvest.ps1 removes PDB files from the harvest
  5. Standard WiX compile (Candle) and link (Light) steps
  6. PostBuild script signs MSI and creates CAB (Release only)
